package: demo.project


SortArray - Array of Integers, derived from JComponent and knows how to shuffle, swap and compare two elements. It can also draw itself.

SortAlgorithm - Abstract class with sort() method.

BubbleSort - Child of SortAlgorithm
SelectionSort - Child of SortAlgorithm 

Sorter - Main driving class which takes sort parameters and visually shows the results
